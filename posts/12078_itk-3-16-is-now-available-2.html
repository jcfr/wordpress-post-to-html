ITK is an open-source, cross-platform system that provides developers with an extensive suite of software tools for image analysis. ITK is distributed under an OSI-Approved BSD License and and has been developed using the same software engineering tools as ParaView (e.g., cmake, dashboards, doxygen). ITK provides leading-edge algorithms for registering and segmenting multidimensional data. Methods exist in the InsightApplications repository for connecting ITK and VTK pipelines together, allowing ITK methods to be easily integrated into ParaView modules.

The ITK development team released the latest version, 3.16, on September 15, 2009. ITK 3.16 is now available for download . More details about this release can be found in the official release announcment , or by going to the ITK Wiki and searching for “Release 3.16”.
