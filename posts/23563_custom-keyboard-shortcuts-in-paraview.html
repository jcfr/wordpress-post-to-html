A new feature coming in ParaView 5.6 is the ability to assign custom keyboard shortcuts to items in ParaView’s menus.  Look under Tools -> Customize Shortcuts… to pull up a dialog that you can use to add or change custom keyboard shortcuts to menu items.


This dialog has a list of items in the menus and their current keyboard shortcut.  Select an item and then press Record followed by the key sequence to set the shortcut.  Specific menu items can be quickly located by typing in the search box at the top.  Shortcuts that are not application defaults will be shown in bold as seen in the image above.
