dockcross: C++ Write Once, Run Anywhere

Want to maximize the portability of your C++ application with minimal effort?   Want to share your application with a broad audience without having to address complex and corner-case installation issues?

Dr. Matt McCormick will be giving a talk that will describe how it is possible with CMake and Docker (via dockcross ) to effectively build native executables for nearly any platform!

When: November 17, 2016, 6:00pm

Where: Triangle C++ Developers Group Meetup
Align, 2121 RDU Center Dr, Suite 200, Morrisville, NC ( map )
