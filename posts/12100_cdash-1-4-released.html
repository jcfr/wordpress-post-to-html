CDash the open-source, web-based software testing server has had its third major release since October. CDash aggregates, analyzes and displays the results of software testing processes submitted from clients located around the world, conveying the state of a software system to continually improve its quality.

The main CDash server for open-source projects is now hosting 12+ projects.

The 1.4 release fixes more than 60 bugs and adds more than 20 new features including:

  • New XML parsing using SAX
  • Support for filters
  • Support for labels on builds, tests, coverage files
  • Introduction of models (MVC framework)
  • Nightly testing of CDash itself:
    http://www.cdash.org/CDash/index.php?project=CDash
  • Support for subprojects (CMake/CTest CVS is required)
    http://trilinos-dev.sandia.gov/cdash/index.php?project=Trilinos
  • Improved navigation
  • Improved database indexing
  • Improved email reporting
  • Email for coverage files
  • Improved column sorting

Personal CDash

If you would like to use CDash for your software project, Kitware is now hosting a CDash server that allows anyone to quickly setup a new project. This exciting, free service allows project to be administered entirely from the CDash web interface. To start submitting to your own personal CDash, register at http://my.cdash.org , and create your project.
