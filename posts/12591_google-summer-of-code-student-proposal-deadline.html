We are taking part in the Google Summer of Code program this year. If you are a student and would like to take part then please get your application in before the deadline tomorrow, on 21 March (19:00UTC), please pass the word on to students who might be interested too. The VTK ideas page shows some of our ideas, or you can suggest your own.

Let us know if you have questions, we look forward to taking part in the tenth Google Summer of Code this year! Our original announcement contains a few more links, and we encourage interested students to get involved early. Learn the tools , build VTK and propose a simple change that demonstrates use of the basic version control and code review tools if you can. Find something you are really interested in, and write a great proposal. There are also many other organizations taking part, and so if you can’t find something for VTK consider some of the other amazing open source projects that are part of this year’s program.
