On Friday, Julien Finet and Stephen Aylward gave a presentation at Duke Visualization Friday Forum titled “3D Slicer, an open-source software framework for visualization and image analysis.”

Description

This is a two-part presentation. The first part discusses the role of Kitware’s high-quality software processes for developing open-source software and running a research lab. The second part discusses the image analysis and visualization capabilities of 3D Slicer , an open-source medical imaging application built using Kitware’s software processes. The presentation also describes how Slicer can be extended to facilitate research and speed-up development.
