ITK 3.12 was released on March 3, 2009. You can download this release from:

  • http://www.itk.org/ITK/resources/software.html
  • through CVS by using the tag:ITK-3-12
  • from Sourceforge: http://sourceforge.net/projects/itk/

The list of new classes and files included in this release can be found at the following Wiki page: http://www.itk.org/Wiki/ITK_Release_3.12

The changes between this release and the previous one can be found at the following Wiki page: http://www.itk.org/Wiki/ITK_Release_3.12_Changed_From_Previous

Please let us know if you encounter any difficulties updating to ITK 3.12 or if you have any questions.
