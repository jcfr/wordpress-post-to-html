The ParaView Catalyst User's Guide version 2.0 has been updated for ParaView 4.3.1. The introduction delves into the reasons for using Catalyst for in situ processing. The other main sections are targeted for users, developers and installers to provide each of them the needed information to properly use Catalyst. Major additions include:

  • A section on ParaView Live for connecting the ParaView GUI to a running simulation code that's instrumented
  • A listing of the available examples and descriptions to help guide instrumenting codes with Catalyst.
  • A section on reusing simulation memory for arrays that doesn't match VTK's expected memory layout.
