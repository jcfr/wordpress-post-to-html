We are pleased to announce the release of Tomviz 1.3.1 ( release page ), this is a bug fix release for the 1.3.0 release announced last week, and fixes several minor issues discovered after the release. These include display of the file name when restoring a pipeline from the state file, a crash showing the color legend if no data was loaded, and the removal of visualizations from the pipeline. We encourage anyone who tried 1.3.0 to upgrade in order to benefit from these bug fixes.

The Tomviz project is developed as part of a collaboration between Kitware and the University of Michigan under DOE Office of Science contract DE-SC0011385. This is a community project, and we are very pleased to take input and contributions from all in the community.
