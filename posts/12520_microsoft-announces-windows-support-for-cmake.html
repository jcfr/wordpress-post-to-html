In a Windows Blog post by Adam Denning, Microsoft recently announced Windows support for CMake. Microsoft’s fork of CMake, CMakeMS, provides developers with the ability to target Windows Store and Windows Phone apps. It is available on CodePlex . According the blog, working with Kitware and the CMake community, Microsoft plans to “incorporate feedback and integrate it soon in the public CMake repository.”

CMake is the cross-platform, open-source build system designed to build, test, and package software. In June, it reached a milestone with the release of version 3.0 . Recently, Kitware announced the availability of CMake 3.0.1 for download. The new release includes several bug fixes, which are described in a previous post on Kitware’s blog . To download the latest version of CMake, please visit CMake’s download page .
